# OBI Protocol

Triggers: `/obi`, `@obi`, `obi sync`, `/obi audit`, `/obi release`.

Follow `~/.obi/protocol/OBI_PROTOCOL.md`. Generate missing project docs,
update active canonical Obsidian docs in place, and write a sync report.

Prefer Obsidian MCP if available. If MCP is unavailable, write to the vault path
from `~/.obi/config.toml`.
