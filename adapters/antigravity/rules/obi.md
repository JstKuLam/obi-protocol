# OBI Rule

Triggers:

- `/obi`
- `@obi`
- `obi sync`
- `/obi audit`
- `/obi release`

Behavior:

- Follow `~/.obi/protocol/OBI_PROTOCOL.md`.
- Generate missing docs from repo analysis.
- Update active canonical docs in Obsidian.
- Use archive only for major scope or architecture changes.
- Write a sync report every time.
